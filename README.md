WrapLayout
=========

介绍
---

自动换行布局，水平排列子项，并自动换行，支持不等长不等宽子项，且可以设置垂直间距与水平间距及子项对齐模式。

使用
---

**引用:**
```
dependencies {
    ...
    implementation 'io.github.alexmofer.widget:wraplayout:2.0.0'
    ...
}
```
